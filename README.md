## Requirements

Node: 'v16.13.1'
Npm: '8.1.2'

## Getting Started

First, install all the dependecies:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Run Tests

```bash
npm run tests
```
